#### Brimko - Social Media Application

Brimko is an open source social media platform for developers across the globe, where they can showcase their acheivements, share ideas, and contribute to open source project.


# Project Setup
  git clone [this repo](https://github.com/Rossville/brimko.git)


# Backend Setup
 - cd brimko/backend
 - npm install
 - npm run dev


# Frontend Setup
 - cd brimko/frontend
 - npm install
 - npm run dev
